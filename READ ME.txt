Ubraha anay ni bag-o mo e serve ang laravel project nga gin clone mo HAHAHAHA

1. E rename ang .env.example nga file sa .env
2. E run ni sa terminal: composer install
3. E run ni sa terminal: npm install
4. E run ni sa terminal: php artisan key:generate
5. E run ni sa terminal: php artisan migrate (type lang yes kung may pamangkot)
6. E run ni sa terminal: php artisan storage:link
