Ubraha anay ni bag-o mo e serve ang laravel project nga gin clone mo HAHAHAHA

1. E copy kag paste ang .env.example nga file tas e rename .env
2. E run ni sa terminal: composer install
3. E run ni sa terminal: npm install
4. E run ni sa terminal: php artisan key:generate
5. E run ni sa terminal: php artisan migrate (type lang yes kung mangkuton ka nga ma ubra sang database)
6. E run ni sa terminal: php artisan db:seed
7. E run ni sa terminal: php artisan storage:link
8. E run ni sa terminal: npm run build
